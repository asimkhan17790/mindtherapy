# MindTherapy 💖

A mental wellness mobile app built with Flutter & Node.js

## 🚀 Quick Start

**Backend:**
```bash
cd backend-nodejs
npm install
npm run dev  # Starts on localhost:3000
```

**Frontend:**
```bash
cd frontend-flutter  
flutter pub get
flutter run -d chrome --web-port 8081
```

## 🏗️ Architecture

- **Backend**: Node.js + Express + MongoDB
- **Frontend**: Flutter (cross-platform mobile)
- **Features**: Mood tracking, positive affirmations, task suggestions

## 📱 Core Features

✅ Daily affirmations  
✅ Mood check-in (😊😐😔)  
✅ Smart task suggestions  
✅ Reminders about loved ones
